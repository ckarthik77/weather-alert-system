About This Project

DevOps CI/CD Laboratory Project

This project demonstrates enterprise-level DevOps practices by implementing a complete CI/CD pipeline for a Spring Boot microservice. The Weather Alert System monitors weather conditions and triggers intelligent alerts when thresholds are exceeded.

The project showcases automated testing, containerization, continuous integration, and deployment automation - core skills required for modern DevOps engineers.
